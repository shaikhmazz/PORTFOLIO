// FEATURED WORK

GeoProteoNet

A non-invasive, two-stage pipeline that screens for fetal anemia from Doppler ultrasound imagery — pairing a custom segmentation network with a classical classifier on the hemodynamic features doctors already trust. Read end to end, it's also a case study in pipeline design: two stages, each with its own success metric, wired together and served through a real API.

The problem

Confirming fetal anemia usually means an invasive procedure. Doppler ultrasound of the middle cerebral artery is a safer, non-invasive alternative — but reading it well depends on an expert sonographer's eye. GeoProteoNet automates that read.

Stage 01 — Segmentation

A custom CNN encoder–decoder locates and segments the artery waveform region in each ultrasound frame, isolating the signal worth measuring from everything around it.

93.99% Dice Coefficient

Stage 02 — Classification

From the segmented waveform, the pipeline extracts five hemodynamic features — PSV, EDV, RI, PI, and S/D ratio — and feeds them to a Random Forest classifier that flags anemia risk.

92.90% F1-score

Serving it

A FastAPI backend wraps the pipeline; a React frontend turns the output into a readable diagnostic report — so the result is usable by a clinician, not just a notebook.
