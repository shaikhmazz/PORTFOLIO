# Portfolio — Shaikh Mazein Ahmed

Personal portfolio site, built as a static page (no framework, no build step) so it can be
hosted directly on GitHub Pages.

**Live site:** add your GitHub Pages URL here once deployed.

## Structure
- `index.html` — all content
- `style.css` — visual design (dark, diagnostic-monitor aesthetic, tied to the GeoProteoNet project)
- `script.js` — scroll-reveal behavior
- `resume.pdf` — add your résumé PDF here (referenced by the "Résumé" button)

## Running locally
Just open `index.html` in a browser. No install, no server required.

## Deploying
See the deployment steps shared alongside this project, or: Settings → Pages →
Deploy from branch → `main` → `/ (root)`.
